# Real Seasons

Real Seasons is a modern Minecraft mod focused on bringing immersive, realistic seasonal visuals into your world without changing how the game plays.

## Philosophy

Real Seasons is built with a simple priority in mind:

* Stay up to date with the latest version of Minecraft
* Visual fidelity over feature bloat
* Zero gameplay impact

This means compatibility with other mods is always secondary to keeping the mod current and stable on modern Minecraft versions.

## Features

### Purely Aesthetic

Real Seasons does not change any gameplay mechanics. There are no effects on crops, weather behavior, mob spawning, or survival systems. Everything is visual.

Because of this design, the mod is safe to use in multiplayer environments and will never crash when connected to a server that does not have the mod installed.

### Per Biome Seasonal Colors

Every biome can have its own seasonal identity.

Forests, swamps, plains, and modded biomes can all shift colors differently across the year, allowing for a more natural and varied world instead of a one size fits all approach.

### Flexible Time Systems

#### Real Time Mode

Seasons can follow real world time, progressing naturally throughout the year.

#### In Game Time Mode

Seasons can also be based on Minecraft days.

* Supports any number of sub seasons
* Each sub season must be at least 5 in game days long
* Fully customizable seasonal pacing

This allows for fine control over how quickly seasons change in your world.

## Design Goals

* Always modern and updated alongside Minecraft
* Realistic seasonal transitions
* No gameplay changes
* Stable in any environment
* Fully aesthetic experience

## Summary

Real Seasons is for players who want their world to feel alive and dynamic without altering the core Minecraft experience. It focuses on visual storytelling through color and time, while staying lightweight, modern, and reliable.
